Hướng dẫn lấy Apps Script URL
1. Mở Google Apps Script → chọn project FullERP
2. Click Deploy → New deployment
3. Type: Web App | Execute as: Me | Who has access: Anyone
4. Click Deploy → Copy URL → Paste vào ô trên
