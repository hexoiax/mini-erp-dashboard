📖 Hướng dẫn Deploy
1. Mở Google Sheet → Extensions → Apps Script
2. Copy file FullERP_AppScript_V2.gs paste vào
3. Deploy → New → Web App → Execute as Me → Anyone
4. Copy URL → Paste vào ô trên → Save

✅ Hệ thống:
→ Dùng Apps Script Cache (tối ưu tốc độ)
→ 1 request = tất cả data (Products, Inventory, Orders, Dashboard)
→ Không cần Google Sheets API Key
