📖 Hướng dẫn cấu hình
1. READ (Google Sheets API):
  → Google Cloud Console → Enable Sheets API
  → Create API Key → Copy vào ô trên

2. WRITE (Apps Script):
  → Mở Google Sheet → Extensions → Apps Script
  → Deploy → New → Web App → Anyone
  → Copy URL → Paste vào ô trên

3. Sheet Structure:
  → Product Master: A=ID, B=Name, C=Code, D=Price, E=Image Link, F=Emoji, G=Sizes, H=Status
  → Inventory: A=SKU, B=Product, C=Size, D=Price, E=Stock, F=Sold, G=Emoji
  → Orders: A=ID, B=Code, C=Customer, D=Phone, E=SKU, F=Qty, G=Total, H=Status
  → Event Log: A=Type, B=Message, C=Payload, D=Timestamp
